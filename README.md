# My Skills

This repository contains local Codex skills.

## Skills

| Skill | Path | Overview |
| --- | --- | --- |
| `git-commit` | `git-commit/` | Reviews Git changes, stages relevant files, chooses a commit message, and commits only when explicitly requested. |
| `plan-from-issue` | `plan-from-issue/` | Fetches a GitHub issue, investigates the codebase, and prepares an implementation plan before work begins. |
| `roleplay-assistant` | `roleplay-assistant/` | Applies a selected Markdown character profile as a lightweight response style while keeping the user's task as the priority. |
