# Project Rules

## Skill Documentation

- When adding a new skill directory with a `SKILL.md`, update `README.md` in the same change.
- Add the new skill to the `## Skills` section in `README.md` with its name, directory path, and a short overview of when to use it.
- Add the new skill to the README skill list with its name, directory path, and a short overview of when to use it.
- Do not commit a new skill addition without the matching README update.
- Do not add new columns to the README skill table.
